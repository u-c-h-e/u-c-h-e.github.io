Why spaceKit?

You’ve met Mike before, right? The one with the desktop full of images folders? He gets lost in a wave of x on his dashboard. He’s got personal, work and x all on the same cloud. It’s easy to upload a file to the dashboard so it’s safe in the Cloud. But he doesn’t always get to organizing things. Maybe you know him. Maybe you are him. It’s all good because help is on the way.

HOUSTON, WE HAVE A PROBLEM

Cloud storage services only offer users a limited amount of features to help with file organization.

Folders are helpful, but I asked the question how could we improve cloud storage cloud storage organization for users that could use a bit of help managing their files?

SOLUTION

Better control wrangling various content

With a better vision of the type of user I wanted to “solve” for, I begun brainstorming the solutions a new cloud storage app should address:

  1. Ensuring that dashboards remain clutter-free;
  2. Encouraging users to organize their content in a helpful and unobtrusive manner;
  3. Establishing an easy-to-understand hierarchy for saving content; and
  4. Providing alternative organizational methods above and beyond folders.

TESTING

Usability testing

In the first set of usability tests users were tasked with signing up for an account, adding a new piece of content to their dashboard, and organizing a file. The second batch was conducted after I created the high fidelity wireframes. Testers were tasked with creating a folder, customizing their folder, and more.

Many users mentioned how much they liked the vibrant colors, the display controls, and how easy to navigate and familial the interface was. The editable folder cover was also a big hit!
